### Key Architecture Decisions
- Frontend reads from `site.json` (loaded at startup)
- Admin writes to SQLite DB
- `syncAllToSiteJson()` syncs DB → site.json after every admin service/policy/testimonial change
- This means: site.json is derived from DB, not the other way around

### Services Sync
