# God-Tier Dev Skills — Full Specification

Uploaded by Dan: 2026-03-30

## Purpose
Complete skill suite for building any digital product from concept to launch.

## The 10 Skills

1. **VisionForge** — Transform rough ideas into market-worthy products
2. **BuildForge** — Full technical architecture and implementation
3. **ExperienceForge** — Premium UX/UI and polish
4. **AdminForge** — Full operator admin panel
5. **UserForge** — Private user area / customer portal
6. **RevenueForge** — Monetization systems
7. **IntegrationForge** — Cross-system coherence
8. **EvolutionForge** — Safe changes and upgrades
9. **ShipForge** — QA and release readiness
10. **OrchestratorForge** — Coordinates all skills as one org

## Global Rules
- Preserve core vision while improving weak areas
- Expand ideas into richer, higher-value systems
- Never generic or shallow work
- Build for real users and real operators
- Every project with users = serious private area
- Every business project = serious admin panel
- Monetization built-in, not bolted on
- Stripe architecture with sandbox/test + live keys + admin switch
- Ad/affiliate/sponsor structures where relevant
- Verify all work before approval
- Coordinate outputs so result feels unified
