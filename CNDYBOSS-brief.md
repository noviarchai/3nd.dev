==================================================
CORE OBJECTIVE
==================================================

Build a COMPLETE production-grade booking backend + CRM platform for wellness providers and appointment-based businesses.

Modular, scalable, API-first, mobile-friendly, white-label, cloneable.

Target industries: massage, acupuncture, lash, beauty, nails, hair, estheticians, wellness clinics, IV therapy, spas, solo practitioners, multi-provider clinics.

Design: Stripe Dashboard / Linear / Notion / Square Appointments / Fresha / Cal.com level. Modern, luxurious, calming, fast, minimal, premium. Dark + light mode. ADHD-friendly, mobile-first.

==================================================
SYSTEM ARCHITECTURE
==================================================

API-first, frontend agnostic. REST API + widgets + embeds + webhooks. PostgreSQL database.

Multi-tenant: each client has isolated data, branding, settings, staff, bookings.

CORE FEATURES
==================================================

1. BOOKING SYSTEM
   - Service selection, category grouping, staff/provider selection
   - Duration management, buffer times, availability logic
   - Time slot generation, recurring schedules, break management
   - Vacation/blocked dates, booking limits, capacity limits
   - Group appointments, multi-service bookings
   - Booking reschedule/cancel, waitlists, timezone support
   - Mobile booking flow, intake forms, notes

2. CRM SYSTEM
   - Client profiles: name, contact, birthday, notes, history
   - Intake forms, tags, loyalty points, spending totals
   - Consent forms, files/images, preferences, allergies
   - SMS/email history, internal notes, activity timeline

3. CALENDAR SYSTEM
   - Day/Week/Month/Staff/Room views
   - Drag/drop, resize, color-coded, conflict detection
   - Google Calendar sync, ICS support

4. STAFF + PROVIDERS
   - Multiple staff, schedules, permissions, services
   - Commission tracking, availability overrides, time off
   - Roles: Owner, Admin, Reception, Provider, Assistant

5. AUTOMATIONS
   - Triggers: booking created/cancelled/rescheduled, no-show, new client, birthday, follow-up
   - Actions: send email/SMS, add tag, notification, webhook, loyalty reward

6. NOTIFICATIONS
   - Email + SMS with custom templates, variables, branding
   - Booking confirmation, reminder, cancellation, follow-up, rebooking, birthday, promotional

7. FORMS + INTAKE SYSTEM
   - Drag/drop form builder, intake forms, consent forms, waivers
   - Conditional logic, file uploads, e-signatures, saved templates

8. PAYMENTS (preparation for)
   - Deposits, partial/full payments, memberships, packages
   - Gift cards, subscriptions, invoices, receipts
   - Stripe, Square, PayPal integrations

9. MEMBERSHIPS + PACKAGES
   - Service packages, session tracking, membership plans
   - Recurring billing, usage limits, rewards, loyalty system

10. MARKETING TOOLS
    - Email/SMS campaigns, promotions, discount codes
    - Referral system, review requests, rebooking reminders

11. ANALYTICS
    - Revenue, bookings, utilization, retention
    - Top services, staff performance, repeat clients
    - Cancellation/no-show rates, client growth

12. API-FIRST SYSTEM
    - Endpoints: Services, Staff, Availability, Bookings, Clients, Forms, Payments, Automations
    - API keys, rate limiting, webhooks, docs, sandbox mode

13. EMBED SYSTEM
    - Embeddable booking widgets, popup, inline, direct booking pages
    - Copy/paste embed code

14. FRONTEND CONNECTION SYSTEM
    - Pull services, pricing, availability, create bookings, submit forms
    - Show providers, display reviews

15. WEBSITE CONTENT CONTROL
    - Editable: service descriptions, prices, FAQs, policies, banners, promotions

16. MOBILE EXPERIENCE

17. PERFORMANCE

18. SECURITY
